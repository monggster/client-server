при входе нужно указать имя

подключиться к комнате или создать если не существует
#enter-room <имя комнаты>

посмотреть существующие комнаты
#show-rooms

удалить комнату
#delete-room <имя комнаты>

изначально кидает всех в общую комнату "парадная"

отправить сообщение
(room-name): <текст>

чужое сообщение
(room-name) @user123: <текст>

выход
#exit








у каждого сообщения есть поле -- имя комнаты

имя отправителя
имя комнаты
текст

комната -- набор пользователей с общим пространством обмена сообщениями.


словарь: имя комнаты : список дескрипторов

имя отправителя
имя комнаты
дескриптор

удобно объединять всех по комнатам, чтобы можно было легко найти всех, кому отправить сообщение

удобно разделять по дескрипторам, чтобы можно было легко найти кого удалить

если имя будет уникальным
группировка по комнатам 
{
	"roomname":
		[
			{
				"username": username,
				"descriptor: descriptor
			},
			...
		],
	...
}
